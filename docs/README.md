## Tips for Good Screenshots

- Use a dark terminal theme
- Set terminal size to ~120x30 for consistent aspect ratio
- Use a distinctive font (Fira Code, JetBrains Mono, etc.)
- Consider recording a GIF with [asciinema](https://asciinema.org/) or [terminalizer](https://www.npmjs.com/package/terminalizer)
